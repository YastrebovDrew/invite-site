Мини-сайт приглашения на свидание

Файлы: index.html, styles.css, script.js.

Открытие:

1. Откройте файл invite-site/index.html в браузере;
2. Или запустите локальный сервер в папке invite-site:

```bash
python -m http.server 8000
# затем откройте http://localhost:8000/invite-site/
```

Описание:
- Кнопка "Нет" убегает при наведении/нажатии.
- Кнопка "Да" запускает эффект конфети.
